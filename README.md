## Frequently Asked Questions

**Why does the app ask for location permission?**
Location is used to calculate accurate sunrise/sunset times, Rahu Kalam, and other Panchanga details specific to your city.

**Why does the app ask for notification permission?**
Notifications are optional and used only to send reminders for festivals, vratas, and poojas you set up yourself.

**Why does the app ask for tracking permission?**
The app shows ads to free users. Allowing tracking enables personalized ads. You can decline — the app works fully either way.

**How do I cancel my subscription?**
On iPhone: Settings → Apple ID → Subscriptions → KalaDarshini → Cancel.
On Android: Play Store → Profile → Payments & subscriptions → Subscriptions → KalaDarshini → Cancel.

**The Panchanga timings seem wrong. Why?**
Make sure location access is enabled so the app can calculate timings for your exact location.
